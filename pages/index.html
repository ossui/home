OSSUI Home Page

What is it? The Open Source Security User Interface (OSSUI) Project is a free-to-use information security software solution based on the open source OSSEC host-based intrusion detection system (HIDS). The focus of this project is to create a HIDS interface that anyone can use and to increase information security awareness. OSSUI features include: alert monitoring for your system, a user-friendly graphical user interface, simplified installation process, and more.



Why do I need it? In today's increasingly interconnected world, cyber-attacks present a real threat to businesses of any size. Cyber attacks can cost businesses thousands for dollars and damages and many companies are not equipped to defend against them. OSSUI is intended for small-businesses who may not have dedicated IT professionals. Therefore, it has been designed with ease-of-use in mind.

How can I get it? OSSUI can be downloaded directly from this site on the Download page. The OSSEC HIDS can be downloaded from OSSEC's site here. Instructions for installation are included with each download.

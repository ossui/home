OSSUI Download & Installation

Supported Systems

  • Ubuntu Linux
  • RHEL
  • Debian Linux

Downloads


Installation Instructions

  1. Download Linux, OSSUI, OSSEC, and Installation Script
  2. Decompress downloaded files using Archive Manager
  3. Replace install.sh script in ossec folder
  4. Right-click Linux desktop and open terminal
  5. Move to ossec folder (e.g. $ cd Downloads/ossec) and open root shell: sudo su
  6. Execute install.sh script: ./install.sh
  7. Choose whether to enable email notifications [Y/N]
  8. From root, enter: /var/ossec/bin/ossec-control start
  9. Execute ossui setup script: /Desktop/ossui/setup.sh
  10. Create username and password for OSSUI
  11. Open browser and navigate to localhost/ossui

Troubleshooting

  • Problem executing setup.sh: Check permissions (chmod +x setup.sh)
  • OSSUI alerts not appearing: Restart Linux or Apache (e.g. # service apache2 restart)
